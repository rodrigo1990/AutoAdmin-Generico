== Description ==

MyThemeShop connect is an automated theme updater that will upgrade <a href="https://mythemeshop.com/themes/">MyThemeShop themes</a> & plugins to their latest versions. The MyThemeShop Connect plugin is a useful plugin to help notify you about updates to MyThemeShop themes & plugins. This plugin is tested and working with all MyThemeShop themes & plugins and on both single and multi-site installations of WordPress.

= Troubleshooting: =

You can reset all the notifications easily by heading over to below URL http://yourdomain.com/wp-admin/index.php?page=mts-connect&reset_notices=1
Replace yourdomain.com with your actual domain name.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `mythemeshop-connect` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can see `MyThemeShop` link in left sidebar of WP Dashboard.
4. Enter your MyThemeShop Username and Password in plugin settings and you are ready to get new theme/plugin updates.

== Frequently Asked Questions ==

= Plugin showing Double Notifications =

You can reset all the notifications easily by heading over to below URL http://yourdomain.com/wp-admin/index.php?page=mts-connect&reset_notices=1
Replace yourdomain.com with your actual domain name.

= Showing Membership Expired Notification =

If you are seeing this notification, that means your MyThemeShop Membership is expired. Please head over to MyThemeShop.com and renew your membership to get updates.

= I have active membership still showing Membership expired notification =

If you are seeing this message, please disconnect from MyThemeShop tab in your dashboard and connect again and if still your are not able to update, please contact us on our Support forum (http://community.mythemeshop.com/)

= I have customized my theme extensively and don't want to update =

You can dismiss the update notification using close button present in notification at the right top corner. It's also advised to use a Child theme if you make any PHP files related changes, so you could still update to the latest version of the theme.
